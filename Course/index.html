Eventos del DOM

Tipos de eventos

En JavaScript existen distintos tipos de eventos que permiten **escuchar y reaccionar a acciones específicas** dentro de una página web. Todos los eventos comienzan con el prefijo `on` seguido del tipo de acción que se desea detectar. Estos eventos se colocan como **atributos en las etiquetas HTML**, y cuando ocurre la acción correspondiente, se ejecuta el código que se haya indicado dentro del atributo, por lo que los siguientes son ejemplos de los tipos de eventos.

onclick="": Este evento **detecta cuando se hace clic** sobre un elemento. Es uno de los eventos más comunes y sirve para ejecutar una función al presionar botones, enlaces, imágenes, etc.

onscroll="": Se ejecuta cuando el usuario **desplaza (hace scroll)** dentro de una sección o sobre todo el documento. Es útil para detectar si el usuario llegó al final de la página, activar animaciones durante el desplazamiento, entre otros usos.

onmouseover="": Este evento se activa cuando el usuario **coloca el cursor del mouse encima** del elemento. Es común en efectos visuales, menús desplegables, y para mostrar información adicional (como tooltips).

onmouseout="": Contrario al anterior, este evento se dispara cuando el cursor **sale del área del elemento**. Se usa para revertir efectos que comenzaron con `onmouseover`, como ocultar un menú o eliminar un efecto de iluminación.

onchange="": Se utiliza en formularios y se activa cuando el **valor de un campo cambia**, como un input, un select o un checkbox. Es útil para validar datos mientras el usuario interactúa con el formulario.

onkeydown="": Detecta cuando el usuario **presiona una tecla** mientras está enfocado en un campo de texto o en el documento. Es ideal para accesos rápidos, validaciones o juegos con teclado.
